# Prismfly GSAP Website

A responsive React/Vite implementation of the supplied Prismfly Figma page, enhanced with GSAP interactions inspired by the GSAP Demo Hub.

## Included motion

- Hero split-line entrance timeline
- Infinite client marquee
- Pinned horizontal services track
- Scroll-velocity title skew
- Cursor-follow case study preview
- Magnetic CTA buttons
- Animated experimentation dashboard
- Floating partnership labels and gradient mesh
- Animated testimonial carousel
- Reduced-motion accessibility fallback

## Run locally

```bash
npm install
npm run dev -- --host 127.0.0.1
```

Build the production bundle with:

```bash
npm run build
```
